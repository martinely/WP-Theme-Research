# HOME SKELETONS — AI Agency Theme (3 Homes)

One design system, three distinct homepages.
Rule: each section must be template-friendly (Elementor). No “fake” dynamic features.

---

## HOME 1 — AI Automation / Integration Agency (Primary)

### H1-01 Header (sticky)
- Nav: Services / Industries (optional) / Case Studies / About / Resources / Contact
- Primary CTA: “Book a discovery call” (or “Talk to an expert”)

### H1-02 Hero (agency variant)
- Eyebrow: “AI Automation & Integration”
- H1: Outcome-led promise (no numbers unless client supplies)
- Subhead: 1–2 lines about end-to-end delivery (strategy → build → integrate → improve)
- CTAs:
  - Primary: Book / Contact
  - Secondary: Explore capabilities
- Proof slot: client logos OR 2–3 proof chips (placeholders)

### H1-03 Capabilities grid (cards)
- 4–6 cards (e.g., Automation, Integrations, AI Agents, Data/Analytics, RPA, Governance)
- Each card: title + 1 sentence + link

### H1-04 Proof strip (logos or metrics)
- Mode A: logo wall OR
- Mode B: outcomes strip (placeholders like “time saved”, “cycle time reduced”)

### H1-05 Methodology stepper (process)
- 4–6 steps (Discover → Design → Build → Integrate → Train → Support)
- Each step: short label + 1 line description

### H1-06 Case studies teaser
- 2–3 case cards:
  - Client/industry (placeholder)
  - Problem → solution → outcome (1 line each)
  - Link to case study

### H1-07 Industries / use-cases mini grid (optional)
- 6–9 small chips/cards (Finance, Healthcare, Logistics, etc.)
- Purpose: self-selection and scannability

### H1-08 Testimonials (lightweight)
- 2–3 quotes slider OR static cards
- Keep short; add role/company placeholders

### H1-09 CTA band (book)
- Headline + 1 sentence
- Primary CTA: Book call
- Secondary: Email link / contact form anchor

### H1-10 Footer

## HOME 2 — AI Micro-SaaS Products

### H2-01 Header (sticky)
- Nav: Product / Solutions / Pricing / Security (optional) / Resources
- Right side:
  - Primary CTA: “Start free”
  - Secondary link: “Sign in”
  - Optional tertiary: “Request demo”

### H2-02 Hero (SaaS variant)
- H1: Clear product category (“AI meeting notes / AI notetaker / …”)
- Subhead: 1–2 lines value + trust microcopy (e.g., privacy stance placeholder)
- CTAs:
  - Primary: Start free
  - Secondary: Request demo
- Proof slot: ratings badges OR customer logos

### H2-03 Feature stack (core benefits)
- 4–6 features in a grid or stacked rows:
  - Auto notes, summaries, action items, search, sharing, analytics (placeholders)
- Include 1 “how it works” mini row (3 steps) if needed

### H2-04 Integrations row (logo strip)
- Meeting + CRM tools placeholders
- Link: “See all integrations”

### H2-05 Trust / Security bar
- Badges placeholders: SOC2 / GDPR / HIPAA (only as placeholders)
- Link: “Security overview”

### H2-06 Use-cases grid (teams/personas)
- 6 cards: Sales / Support / Product / Recruiting / Marketing / Leadership (examples)
- Each card: 1 line benefit

### H2-07 Testimonials / ratings strip
- 3 quotes OR ratings tiles (lightweight)

### H2-08 Pricing preview (on-home)
- 2–4 plans with monthly/annual toggle visual
- Enterprise: “Contact sales”
- CTA: “Compare plans” → pricing page

### H2-09 FAQ teaser (optional)
- 6 Qs max (accordion)
- Focus: privacy, integrations, pricing, support

### H2-10 Final CTA
- Headline + “Start free” button

### H2-11 Footer (include legal + privacy links)

## HOME 3 — AI Trainers / Consultants

### H3-01 Header (sticky)
- Nav: Programs / For Teams / Curriculum / Instructors / Resources / Contact
- Primary CTA: “Request training plan” (or “Book a briefing”)

### H3-02 Hero (training variant)
- H1: “AI training for teams” (or similar)
- Subhead: Outcomes + practical format (workshops/bootcamps)
- CTAs:
  - Primary: Request plan / Book briefing
  - Secondary: View programs
- Proof slot: corporate client logos OR learner metrics placeholders

### H3-03 Programs grid
- 3–4 program cards:
  - Executive briefing
  - Team bootcamp
  - Role-based workshops
  - Optional: Certification track
- Each card: duration placeholder + target audience + CTA

### H3-04 Outcomes panel
- 3–5 metrics placeholders (hours saved, adoption, confidence)
- Keep non-specific unless user provides stats

### H3-05 Curriculum modules (tabs/accordion)
- 4–6 modules:
  - GenAI fundamentals, Prompting, Agents, Automation workflows, Governance, Evaluation
- Each module: bullets (3 max)

### H3-06 Role / industry tracks (selector)
- 6–8 track cards (Marketing, HR, Ops, Sales, Product, Engineering, etc.)

### H3-07 Instructor bios
- 2–4 profiles with credentials placeholders

### H3-08 Proof / testimonials
- Corporate testimonial OR short case snippet
- Optional: logos row

### H3-09 Pricing (optional logic)
- Mode A: “Request a quote” panel (default)
- Mode B: subscription tiers (only if user has SaaS-like training)

### H3-10 CTA band
- Primary: Request training plan
- Secondary: Email/contact form

### H3-11 Footer

## Shared constraints
- No invented stats, awards, certifications.
- “Security/compliance” elements are placeholders unless explicitly sourced.
- Keep each home scannable: 8–11 sections max (excluding header/footer).
